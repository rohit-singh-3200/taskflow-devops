# TaskFlow DevOps Pipeline

A production-ready Flask task management application built to demonstrate modern DevOps practices, including containerization, CI/CD, and cloud deployment.

## Features

- User Authentication
- Password Hashing
- Session Management
- CRUD Operations
- MySQL Database
- Environment Variables
- Responsive UI

## Tech Stack

- Python
- Flask
- MySQL
- HTML
- CSS
- Git
- GitHub

